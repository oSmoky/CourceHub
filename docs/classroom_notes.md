# CourseHub Classroom Notes

## Core Idea

CourseHub is a small database-driven online course platform. It focuses on the main educational workflow: instructors publish courses and lessons, while students enroll and track their lesson completion.

## Included Modules

- User Management: registration, login, logout, role-based access.
- Course Management: instructor CRUD for course records.
- Lesson Management: instructor CRUD for lesson records under a course.
- Enrollment Management: student enrollment in available courses.
- Progress Tracking: lesson completion status and progress percentage.

## Excluded Features

- Online payments.
- Certificates.
- Recommendation engines.
- Live video conferencing.

## Main CRUD Mapping

- Create: users, courses, lessons, enrollments, progress records.
- Read: course catalog, lesson details, enrollment dashboard, progress status.
- Update: course details, lesson details, completion status.
- Delete: courses and lessons by their instructor.
